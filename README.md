אמריקה הבריטית: שורשים, אידאולוגיה ואטימולוגיה 🧭
​ברוכים הבאים לפרויקט המחקר ההיסטורי והלשוני המקיף על 13 המושבות האמריקאיות ומושבת פלימות'.
​פרויקט זה אינו מסתפק בציון תאריכים ועובדות יבשות. הוא נועד לצלול לעומק ההיסטוריה, האידאולוגיה (דתית, כלכלית או פוליטית) שעמדה מאחורי הקמת כל מושבה, ובעיקר – ניתוח אטימולוגי מפורט ומלא של שמות המושבות תוך הפרדה ברורה בין מרכיבי השם (לטינית, שפות ילידיות, או שמות אישים בריטיים).
​🌟 מאפייני הפרויקט (Features)
​מחקר אטימולוגי מעמיק: כל מושבה מקבלת פירוק של שמה לגורמים, כולל משמעויות ומקורות השם.
​הקשר היסטורי ואידאולוגי: שנות הקמה, זהות המייסדים, והסיבות המדויקות להקמת המושבה.
​ארכיטקטורת רכזת וחישורים (Hub & Spoke): דף בית מרכזי המוביל לדפי מחקר נפרדים לכל מושבה, כדי למנוע עומס ויזואלי ולאפשר פירוט מקסימלי ללא פשרות.
​ציר זמן היסטורי: רכיב אינטראקטיבי המציג את כרונולוגיית ההקמה מ-1607 ועד 1732.
​חווית משתמש מודרנית וחלון צף (Modal): מידע אודות הפרויקט מוגש בצורה אלגנטית מבלי לעזוב את דף הבית.
​💻 טכנולוגיה ועיצוב (Tech Stack)
​HTML5: מבנה סמנטי ונקי.
​Tailwind CSS: עיצוב רספונסיבי ומודרני באמצעות CDN.
​Dark Mode מושלם: תמיכה מובנית וידנית במצב כהה לקריאה נוחה בלילה, עם פלטת צבעים מותאמת אישית (כחול-זהב).
​📱 אופטימיזציה למסכי פרימיום: הממשק תוכנן במיוחד לחוויית קריאה חלקה ונוחה על גבי Samsung S25 Ultra ומסכים גדולים נוספים, תוך הקפדה על פקדי מגע (Touch-manipulation) נוחים, ריווחים נכונים וטיפוגרפיה ברורה.
​Phosphor Icons: לשימוש באייקונים וקטוריים חדים ונקיים.
​📂 מבנה המאגר (Repository Structure)
​images/ - תיקיית תמונות המיועדת לסמלים, דגלים ומפות היסטוריות.
​index.html - דף הבית המרכזי (Hub), הכולל את רשת המושבות וציר הזמן ההיסטורי.
​README.md - קובץ הסבר זה.
​[שם_המושבה].html - דפי מחקר מפורטים לכל מושבה בנפרד (לדוגמה: virginia.html, plymouth.html).
​✒️ מתודולוגיית המחקר
​כל דף מחקר בפרויקט נבנה בקפידה וכולל:
​רקע המייסדים והילדות (היכן שרלוונטי).
​סיבת ההקמה (אידאולוגיה/דת/כלכלה).
​בחירת מיקום המושבה (השיקולים הגיאוגרפיים, האסטרטגיים והסביבתיים).
​טבלה אטימולוגית מפורטת עם פירוק מלא של השם.
​שנות פעילות/הקמה ויעדים.
